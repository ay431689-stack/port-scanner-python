Python Port Scanner

A Python-based TCP Port Scanner that efficiently scans a target host for open ports within a specified range. This project demonstrates socket programming, multithreading, DNS resolution, and basic networking concepts.

Features

- TCP Port Scanning
- Custom Port Range Support
- Multithreaded Scanning
- Domain Name Resolution
- Open Port Detection
- Service Identification
- Error Handling
- Colored Console Output

Technologies Used

- Python 3
- Socket Programming
- Threading
- Colorama

Installation

1. Clone the repository:

git clone https://github.com/your-username/port-scanner-python.git

2. Install dependencies:

pip install -r requirements.txt

Usage

Run the program:

python port_scanner.py

Enter:

- Target IP address or domain
- Port range (Example: 1-1000)

Example

Enter target IP or domain: example.com
Enter port range: 1-1000

Educational Purpose

This project was developed for learning networking and Python programming concepts.
